# Strength Training for Runners

Ash provides runner-specific strength training to prevent injuries, improve power production, and build structural resilience. The strength pillar complements running by targeting muscles, tendons, and movement patterns critical for running performance.

## Philosophy

**Running-Focused**: Every exercise serves running performance or injury prevention
**Equipment-Flexible**: Primarily bodyweight, with optional resistance bands/dumbbells
**Movement Patterns Over Muscle Isolation**: Focus on functional movements (single-leg stability, hip strength, core stability)

## Phased Implementation

### Phase 1: Prescribed Bodyweight Sessions (MVP)

**Goal**: Get runners doing strength work consistently without overwhelming them.

#### Prescription Approach
*   **Session-Based**: Ash prescribes 2-4 bodyweight exercises per strength session
*   **Runner-Specific**: All exercises target running-relevant movement patterns
*   **Simple Tracking**: Done/Not Done checkbox for the entire session
*   **Frequency**: 2-3 sessions per week based on goal type and pillar priority

#### Example Session
```
Strength Session - Lower Body Focus (20 min)
‚úì Complete this session

1. Single-Leg Glute Bridge - 3x10 each side
2. Bulgarian Split Squat - 3x8 each side
3. Plank - 3x30 seconds
4. Clamshells - 2x15 each side
```

#### Benefits
*   **Low friction**: Users don't need to think about what exercises to do
*   **Consistency**: Simple checkbox encourages completion
*   **Education**: Users learn runner-specific exercises
*   **Foundation**: Builds habit before adding complexity

### Phase 2: Exercise Library & Detailed Tracking

**Goal**: Enable users to track individual exercises and progression.

#### Features
*   **Exercise Library**: 50-100 curated runner-specific exercises
*   **Detailed Logging**: Track sets, reps, weight (optional), RPE per exercise
*   **Flexible Sessions**: Users can log prescribed sessions OR custom gym workouts
*   **Progression Tracking**: See strength improvements over time

#### Exercise Library Structure
Each exercise includes:
*   **Name**: Single-Leg Romanian Deadlift
*   **Movement Pattern Tags**: Hip Hinge, Single-Leg, Unilateral
*   **Muscle Group Tags**: Glutes, Hamstrings, Core
*   **Equipment**: Bodyweight, Dumbbell, Kettlebell, Barbell
*   **Difficulty**: Beginner, Intermediate, Advanced
*   **Instructions**: Text + optional video/image
*   **Running Benefit**: "Strengthens posterior chain for powerful push-off and injury prevention"

#### Tracking Interface

#### Benefits
*   **Detailed progression**: See weight/rep increases over time
*   **Flexibility**: Works for both prescribed sessions and custom gym workouts
*   **Data richness**: Enables Phase 3 intelligence features

### Phase 3: Intelligence Layer

**Goal**: Use comprehensive tracking data to provide insights and optimize training.

#### Gap Detection
Analyze training history to identify imbalances:
*   **Movement Pattern Gaps**: "You've done 8 squat-pattern exercises but 0 hip hinges this month"
*   **Muscle Group Imbalances**: "Quad-dominant training detected. Add hamstring work."
*   **Pillar Neglect**: "No strength sessions in 10 days. Your running volume is high - injury risk increasing."

#### Correlation Analysis
Find patterns between training and outcomes:
*   **Injury Correlations**: "Your knee pain appears 2 days after heavy squats + long runs in the same week"
*   **Performance Correlations**: "You run faster on weeks with 2+ strength sessions"
*   **Recovery Patterns**: "Heavy leg days require 48h before quality running sessions"

#### Smart Scheduling
Use complete training picture to optimize scheduling:
*   **Interference Prevention**: "You did heavy squats yesterday, so today's long run is moved to tomorrow"
*   **Balance Optimization**: "You've hit all movement patterns this week except core - here's a 10min core session"
*   **Volume Management**: "Your combined running + strength volume is 15% above baseline - suggesting recovery day"

#### Progression Insights
*   **Strength Trends**: "Your single-leg squat strength has improved 40% since starting this plan"
*   **Training Balance**: "You're running more volume but doing less strength - let's rebalance"
*   **Predictive Warnings**: "This training load pattern preceded your last injury - consider a recovery week"

## Core Exercise Categories

### 1. Single-Leg Stability (Highest Priority)
Running is a series of single-leg movements. These exercises build balance, stability, and running-specific strength.

*   Single-Leg Squats / Pistol Squats
*   Single-Leg Romanian Deadlifts (RDLs)
*   Bulgarian Split Squats
*   Single-Leg Glute Bridges
*   Single-Leg Balance Holds
*   Step-Ups with Knee Drive

### 2. Hip & Glute Strength (Injury Prevention)
Strong hips and glutes prevent common running injuries (IT band syndrome, runner's knee, hip pain).

*   Glute Bridges (double and single-leg)
*   Clamshells
*   Lateral Band Walks
*   Hip Hikes
*   Fire Hydrants
*   Donkey Kicks

### 3. Core Stability
A strong core improves running economy, posture, and power transfer.

*   Planks (front, side, variations)
*   Bird Dogs
*   Dead Bugs
*   Pallof Press
*   Mountain Climbers
*   Russian Twists

### 4. Plyometrics (Power & Elasticity)
Build explosive power and improve muscle/tendon elasticity for faster, more efficient running.

*   Pogo Hops
*   Single-Leg Hops
*   Box Jumps / Step-Ups
*   Broad Jumps
*   Skater Hops
*   Jump Squats

### 5. Posterior Chain (Hamstrings & Lower Back)
Balance quad-dominant running with posterior chain strength.

*   Romanian Deadlifts (RDLs)
*   Good Mornings
*   Hamstring Curls (Nordic, Stability Ball)
*   Reverse Hyperextensions
*   Superman Holds

## Integration with Running

### Scheduling Rules
1. **48-Hour Recovery**: Heavy lower body strength sessions require 48h before key running workouts (intervals, tempo, long runs)
2. **Same-Day Stacking**: Light strength can be done same day as easy runs (run first, strength after)
3. **Taper Protocol**: During race taper, reduce strength volume by 40-50% but maintain intensity

### Frequency by Goal Type
| Goal Type | Strength Frequency | Focus |
|-----------|-------------------|-------|
| **Distance Milestone** | 2x/week | Injury prevention, basic strength |
| **Time Performance** | 2-3x/week | Power production, running economy |
| **Event (Peak Phase)** | 1-2x/week | Maintenance only during taper |
| **Maintenance** | 2-3x/week | Primary focus, build strength base |
