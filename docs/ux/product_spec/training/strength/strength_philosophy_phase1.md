# Strength Training Philosophy - Phase 1 (MVP)

**Purpose**: Provide AI with prescription logic for runner-specific strength training using minimal, bodyweight-focused exercises that complement running without requiring gym access or complex programming.

---

## Core Philosophy

**Running-Focused**: Every exercise serves running performance or injury prevention  
**Minimal & Accessible**: 2-3 bodyweight exercises per session, no equipment required  
**Low Friction**: Designed to supplement existing routines, not replace them  
**Movement Patterns Over Isolation**: Focus on functional movements (single-leg stability, hip strength, core stability)

---

## Intensity Prescription

**Reps In Reserve (RIR)** approach:
- **Moderate (2-3 RIR)**: Most sets, sustainable effort
- **Hard (0-1 RIR)**: Occasional use for key sets

**Rep Ranges**:
- Strength exercises: 8-12 reps
- Core exercises: 30-45 seconds (planks) or 10-15 reps
- Plyometrics: 6-8 reps (quality over quantity)

**Sets**:
- High priority: 3 sets per exercise
- Medium priority: 2-3 sets per exercise
- Low priority: 2 sets per exercise

---

## Scheduling Rules

### **Coordination with Running**

**48-Hour Recovery Rule**: Heavy lower body strength requires 48h before key running workouts (intervals, tempo, long runs)

**Same-Day Stacking**: Light strength can be done same day as easy runs
- Run first, strength after
- Or strength in morning, easy run in evening

**Never Stack**: Don't do hard leg strength + hard running on consecutive days

### **Weekly Template Examples**

**Medium Priority (2-3x/week)**:
- **Monday**: Rest
- **Tuesday**: Intervals
- **Wednesday**: Easy Run + Strength (Upper Body + Core)
- **Thursday**: Easy Run
- **Friday**: Strength (Lower Body Focus)
- **Saturday**: Long Run
- **Sunday**: Rest or Strength (Full Body)

**Low Priority (1-2x/week)**:
- Fit in 1-2 sessions on easy run days or rest days
- Focus on injury prevention exercises (single-leg, hip/glute)

---

## Recovery Weeks

Every 4 weeks, reduce strength volume:
- Drop to 1-2 sets per exercise (vs. 3 sets)
- Maintain intensity (same difficulty)
- Aligns with running recovery blocks

---

## Key Takeaways for AI

- **Keep it simple**: 2-3 exercises, bodyweight only
- **Rotate weekly**: Follow the 4-week rotation pattern
- **Prioritize single-leg**: Single-leg exercises are highest priority for runners
- **Schedule smart**: 48h before key runs, can stack with easy runs
- **Reduce in taper**: Drop to 1-2x/week, reduce volume 40%
